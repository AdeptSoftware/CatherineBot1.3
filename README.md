Бот "Catherine" для чатов вк [ver. 1.3 w.2].

Использует DialogFlow, YandexDiskAPI.

Содержит команды для работы с API Vainglory и развлекательные команды для участников бесед.


Автор: Adept

Связь: AdeptSoftware@yandex.ru

Группа: https://vk.com/catherinebot (changelog и описание команд)

P.S. Готово для запуска на heroku.com

Upd.: Проект разрабатывался с целью изучить возможности языка и улучшить свои компетенции в программировании, а также для завлечения игроков в гильдию по игре. Так как чат с таким ботом позволял (пока не прикрыли возможность) обращаться к API игры для получения от нее информации о матчах и другой информации.
